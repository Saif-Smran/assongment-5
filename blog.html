Question-1: What are the different ways to select an element in the DOM?

Answer: There are multiple ways to select an element in the DOM:

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent?

Answer: These properties retrieve or update content inside an element, but work differently:

  • innerHTML: Returns or sets **HTML content**, including elements.
  • innerText: Returns only **visible text**, ignoring hidden content.
  • textContent: Returns **all text**, including hidden content.

Question-3: What is event delegation in the DOM?

Answer: Event delegation allows handling events **efficiently** by attaching a listener to a **parent element**, instead of adding individual listeners to multiple child elements.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling means that an event triggered on a **child element** propagates **upward** to its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: You can dynamically manage elements using JavaScript:

  • document.createElement("tag") → Creates an element.
  • parent.appendChild(childElement) → Adds a child to a parent.
  • parent.removeChild(childElement) → Removes a child from a parent.
